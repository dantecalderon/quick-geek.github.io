Back in 2015, Google caused dissatisfaction with some developers, having obtained the domain zone .dev in its sole use. Now the situation is changing , and the opportunity to buy a domain gets any.

Well, right now, not quite any: from February 19 to 28, the Early Access Program is operational with an additional fee for registering a domain, reaching up to $ 11,000. But every day the amount will decrease, and from February 28, only the standard payment of $ 12 / year will remain, so it will be suitable not only for large companies, but also for small personal developers about development.

A number of companies are already active in the new zone (and it doesn’t even simply mean “set up a redirect”). Here are some examples:

  • web.dev and opensource.dev - sites directly from Google
  • github.dev - GitHub here does not redirect to its main page, but offers developers to file a personal website using GitHub Pages
  • ycombinator.dev - the famous startup accelerator YCombinator gathered together links to its materials for developers
  • slack.dev - seems to have made a separate site from the page of the main site api.slack.com/tools
  • jetbrains.dev - in fact, a landing page for JetBrains products that you probably already know
  • codecademy.dev - looks like the usual main Codecademy , but without the “Upgrade to Pro” button. Trimmed version with only free courses?
  • women.dev - Women Who Code
  • mdn.dev - Mozilla's MDN Web Docs

For addresses like microsoft.dev or apple.dev, nothing opens, but you cannot get them. But habr.dev is free for now!

Like the previously opened zone .app, .dev was originally designed for HTTPS: browsers that support HSTS (including all popular ones) will access all .dev sites via HTTPS only.

Previously, addresses with the end of .dev were sometimes used by developers to test their products, but in 2017 there were problems with the appearance of a domain zone.
