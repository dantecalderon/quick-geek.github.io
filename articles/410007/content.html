CONNECT

It consists of utilities assembled in an attempt to establish a connection between a Compaq laptop without a network card and a large computer, after I tried to install Windows 95 using the only diskette found in the bins. This is LapLink (it still exists, the problem of data migration between devices has not been resolved) and Symantec PC Anywhere (development stopped, in 2012 there was a massive source leak with a serious risk to users). The latter provides several features familiar to the modern user: copying files over the network, global and local, via a direct cable, remote desktop, sharing network access.


With it, I tried to connect to another laptop, older, which does not have a cdrom, and I do not have a single floppy disk, no cables and network PC-cards (there are all these, but 32-bit, and 16 are needed). To do this, you can use the non-obvious, but universal infrared port in older laptops. Alas, it does not work very quickly, but with PCAnywhere on top it is also not very reliable.
