The main difficulty in solving such problems with connecting to Siebel is that the "disease" is asymptomatic. If the file system is occupied by an antivirus, then waiting for a response, the user can safely go to drink coffee, meditate or even take a nap for an hour or two ...

Then we look at how fast the network works, whether there is access to the file system - it may be that it moved to another storage system with different network settings.
