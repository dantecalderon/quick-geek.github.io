Statistics for file types in the Wildfire cloud sandbox

Well, we saw how many applications transfer files. And what files and which ones are malicious?

The following statistics are collected by file types. It can be seen that most of the files that go to check are PDF, but most of the file types that contain malicious code are EXE. It does not mean at all that you only need to protect against EXE, because one malicious file for MacOS can be enough for you to lose all data.
