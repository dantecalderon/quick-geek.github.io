Microsoft cut WMP functionality in Windows 7, leaving it the same in Windows 8 and 10



Microsoft continues to promote its Windows 10 operating system, doing everything to persuade users to leave the previous OS and use the latest version. Some company actions raise questions.

The other day it became known that the corporation cut the functionality of the media player in Windows 7, we are talking about Windows Media Player. Changes do not affect file management. The player is no longer able to display information about video and audio, respectively, and sorting by artist, album, genre, etc. is not available.

The company itself explains this by analyzing user feedback, after which it decided to disable the metadata processing service in Windows 7. A rather strange explanation, despite the fact that in Windows 10 and Windows 8, WMP functions fully, working with metadata is supported without problems.
